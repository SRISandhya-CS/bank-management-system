# 🏦 Bank Management System

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![OOP](https://img.shields.io/badge/OOP-Concepts-blue?style=for-the-badge)
![HashMap](https://img.shields.io/badge/HashMap-Collections-green?style=for-the-badge)

A **Java console application** for managing bank 
accounts with deposits, withdrawals, transfers, 
and transaction history using OOP and HashMap.

---

## ✨ Features

- ✅ Create new bank accounts (Savings/Current)
- ✅ Deposit money
- ✅ Withdraw money (with balance check)
- ✅ Transfer money between accounts
- ✅ Check account balance
- ✅ View full transaction history
- ✅ View all accounts in formatted table
- ✅ Search account by account number
- ✅ Auto-generated account numbers (ACC001, ACC002)
- ✅ Real-time date and time on transactions

---

## 🛠️ Tech Stack

- **Language**: Java 25
- **Concepts**: OOP, HashMap, ArrayList, 
  LocalDateTime, Collections Framework
- **IDE**: VS Code / IntelliJ IDEA
